
eatSmartly V2.2 Installation – Quick Guide

⚠ Minimum Requirements:
- Operating System: Windows 10 (64-bit)
- Processor: Intel Core i3 or higher
- RAM: 4 GB minimum
- Storage: 128 GB SSD with 15Gb free on D: partition
- Display: Touchscreen PC recommended
- Network: Local LAN / Wi-Fi for QR Menu
- Printer: ESC/POS receipt printer (optional)
- Supported Environment: PHP 8.1 and MySQL 8.0

⚠ Important:

- Ensure you have Administration rights on your Win 10. 
- Ensure you have a D: partition with at least 15 GB free.
- Run all installers as Administrator.

Steps:

1) Install Dependencies

- Run Step_1_eatSmartly_Dependencies.exe as Administrator.
- Wait until installation completes.

2) Install WampServer

- Run Step_2_wampserver3.3.0_x64.exe as Administrator. 
- Install to D:\wamp64 path then, let avery thing else on default settings.
- When finished Start WampServer (pink “W” icon → tray icon must turn green).
- Switch PHP to version 8.1.13. (Left-click the green Wamp tray icon → PHP → Version → 8.1.13 )
- Wait again until tray icon is green.
  Remark: This is a one-time step. After eatSmartly is fully installed, 
  WampServer will auto-start with Windows. You do not need to launch it manually.

3) Install Configuration

- Run Step_3_eatSmartly_Configuration.exe as Administrator with (default settings).
- Launch eatSmartly from desktop shortcut.
- On first run, complete License Activation (send code → enter activation key).

✅ Installation finished. eatSmartly is ready to use.
